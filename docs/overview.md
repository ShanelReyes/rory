# 🏛️ System Overview

The Rory platform is a cloud-native solution for **Post-Quantum Privacy-Preserving Data Mining (PPDMaaS)**. It enables secure analysis on encrypted data through a distributed architecture composed of specialized nodes.

## ⚙️ Core Interaction Model
The platform maps traditional entities into three functional cloud-native components:

* **Client:** Data preparation and local encryption.
* **Manager:** Central orchestration and bucket management.
* **Worker:** High-performance execution engine for secure mining.
